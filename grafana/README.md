# SignalTrace Grafana Dashboards

Two dashboard variants are included — one for each supported datasource approach.

| File | Datasource | Best for |
|---|---|---|
| `signaltrace_overview_infinity.json` | Infinity HTTP plugin | Any deployment, remote Grafana |
| `signaltrace_overview_sqlite.json` | SQLite plugin | Grafana on same host as SignalTrace |

For full setup instructions including plugin installation, datasource configuration, and alerting examples see the [Grafana Integration wiki page](https://github.com/veddegre/signaltrace/wiki/Grafana-Integration).
